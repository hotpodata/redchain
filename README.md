#RedChain

RedChain is a simple daily goal tracker written in Kotlin.

##Features
* Unlimited chains!
* Notifications
*  Daily deminders & Broken chain notifications
* Chain customization
*  Custom Colors
*  Custom Titles
*  Chain by chain notification settings.
* Stats for each chain and globally.
